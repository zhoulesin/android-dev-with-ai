# 图片目录说明

本目录用于存放教程章节配图和证据截图，按章节分目录管理。

## 目录结构

```
docs/images/
├── 00-全局/
├── 01-背景与趋势/
├── 02-大模型选型/
├── 03-Coding工具对比/
├── 04-效率神器Skills与MCP/
├── 05-高赞Skills对比/
├── 06-Skills筛选策略/
├── 07-接入项目全流程/
├── 08-常见陷阱与解决方案/
├── 09-效率与准确率保障/
├── 10-Token经济学/
├── 11-持续更新/
└── 12-团队治理与安全合规/
```

## 命名建议

- 同一章节内按主题命名，例如：`prompt-compare-v1.png`
- 图表建议保留源文件（如 `*.drawio`、`*.excalidraw`）便于后续更新
- 若图片包含敏感信息，提交前先脱敏
